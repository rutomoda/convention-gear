# Convention Gear

3D prints for keyboard conventions and showcases.

## Products

### KURD - Card Stand

A compact, 3D printable card stand designed for DIN A6 sized cards. Features a friction-fit assembly system requiring no glue or hardware. Can be printed as individual parts for multi-color printing or as a complete framed assembly for convenient one-shot printing.

See [KURD/README.md](KURD/README.md) for details.

### RIUS - Mirror Keyboard Display Stand

3D printable keyboard stand with dual mirror plates for displaying custom mechanical keyboards at conventions. Shows the bottom of the keyboard from a wide array of viewing angles. Requires custom-cut 3mm acrylic mirror plates and offers two versions: print-in-place springs or TPU barrel inserts.

See [RIUS/README.md](RIUS/README.md) for details.
