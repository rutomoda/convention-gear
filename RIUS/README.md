# RIUS - Mirror Keyboard Display Stand

3D printable keyboard stand with dual mirror plates for displaying custom mechanical keyboards at conventions. Shows the bottom of the keyboard from a wide array of viewing angles.

The assembly consists of 4 main parts total (and 8 accessory parts for the TPU_BARREL-version): 

- 2x 3D printed side pieces (mirrored)

- 1x mirror back plate

- 1x mirror bottom plate.

## Required Components

### Mirror Plates (Order Custom Cut)

- **Material:** 3mm thick acrylic/PMMA mirror
- **Back plate:** 122mm tall
- **Bottom plate:** 166mm tall
- **Width (both plates):** At least 11mm smaller than your keyboard width

### 3D Printed Parts

**Choose one version:**

#### Version 1: RUIS.stl (Print-in-Place Springs)

- Print 1x as-is and 1x mirrored
- **Warning:** Springs remain under load causing material creep and loosening over time

#### Version 2: RIUS_TPU_BARREL.stl (TPU Barrel Inserts)

- Print 1x as-is and 1x mirrored
- Print 8x RIUS_TPU_BARREL_BARREL.stl in TPU (4 barrels per side piece)

## Manufacturing Instructions

1. **3D Print:**
   
   - Designed for printing without supports
   - Print flat on the side with the pattern opposite the plate slots
   - Print one side piece in chosen version
   - Mirror and print second side piece
   - If using barrel version: Print 8 barrels in TPU (originally designed with 98A)

2. **Order Mirror Plates:**
   
   - Measure your keyboard width
   - Order both plates with width = keyboard width - 11mm minimum 

3. **Assembly:**
   
   - Insert mirror plates into side piece slots
   - If using barrel version: Press TPU barrels into insert bores (4 per side)
   - Place keyboard on top

## Files

- `RUIS.stl` - Side piece with print-in-place springs
- `RIUS_TPU_BARREL.stl` - Side piece with barrel inserts
- `RIUS_TPU_BARREL_BARREL.stl` - TPU barrel insert (print 8x)
